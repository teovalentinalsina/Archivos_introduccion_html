HTML: Hypertext Markup Language

¿Qué es HTML?

Es un lenguaje compuesto por etiquetas las cuales desacriben elementos.

El navegador web es el encargado de interpretar el codigo y mostrarlo de forma comprensible a nuestros usuarios.

Elementos HTML y semántica

Son bloques de construccion para la maqueta de nuestra pagina web. Los elementos HTML nos permiten incorporar texto, imagenes, vinculos, entre otros. HTML tambien proporciona un contexto semantico de elementos el cual describe su proposito, como por ejemplo titulos, parrafos, listas, enlaces, citas y otros.

CSS, Cascade Styling Sheet

Hojas de estilo en cascada

Css es el lenguaje que nos permite agregarle estilos a nuestro documento HTML. CSS describe cómo los elementos HTML deben ser vistos en el navegador. En este link podemos acceder al manual oficial de CSS
